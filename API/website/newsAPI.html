byTitle

The byTitle will return a news with a title match or containing the query
This can be combined with any other parameters as well.

Values
It accepts a string
Use html entities for spaces,ampersands...etc
Example REST Call
HTTP://API_HOME_URL/news.php?msg=query&byTime=News&nbsp;Title

byMonth

The byMonth parameter returns all news that are in that month.
This can be combined with byYear

Values
The byMonth parameter accepts values 1-12 for the corresponding month
Example REST Call
HTTP://API_HOME_URL/news.php?msg=query&byMonth=9

byYear

The byYear parameter returns all news that are in that Year.
This can be combined with byMonth.

Values
It accepts values any that can be considered a year
Example REST Call
HTTP://API_HOME_URL/news.php?msg=query&byYear=2015

byDate

The byDate parameter returns all news that match the exact date given.
The search can modified further by getting the lower or upper bound by using timeframe

byDate Values
Values
It accepts a date in a YYYY-MM-DD(2015-09-19) format
Example REST Call
HTTP://API_HOME_URL/news.php?msg=query&byDate=2015-09-19
timeframe Values
Values
It accepts either future or past. Where future will return all news past the date and past will return all news before the date
Example REST Call
HTTP://API_HOME_URL/news.php?msg=query&byDate=2015-9-19&timeframe=future
