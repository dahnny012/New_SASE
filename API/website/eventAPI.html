byMonth

The byMonth parameter returns all events that are in that month.
This can be combined with byYear

Values
The byMonth parameter accepts values 1-12 for the corresponding month
Example REST Call
HTTP://API_HOME_URL/event.php?msg=query&byMonth=9

byYear

The byYear parameter returns all events that are in that Year.
This can be combined with byMonth.

Values
It accepts values any that can be considered a year
Example REST Call
HTTP://API_HOME_URL/event.php?msg=query&byYear=2015

byDate

The byDate parameter returns all events that match the exact date given.
The search can modified further by getting the lower or upper bound by using timeframe

byDate Values
Values
It accepts a date in a YYYY-MM-DD(2015-09-19) format
Example REST Call
HTTP://API_HOME_URL/event.php?msg=query&byDate=2015-09-19
timeframe Values
Values
It accepts either future or past. Where future will return all events past the date and past will return all events before the date
Example REST Call
HTTP://API_HOME_URL/event.php?msg=query&byDate=2015-9-19&timeframe=future

byTime

The byTime parameter returns all events that match the given time.
This can be combined with any other parameters as well.

Values
It accepts a time value as 24 hour clock format(HH:MM:00)
Example REST Call
HTTP://API_HOME_URL/event.php?msg=query&byTime=23:12:00
